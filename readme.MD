```GET https://chiquito-ipsum.netlify.app/<PARAGRAPHS>/<MODE>```

(optional) paragraphs |1        | How many paragraphs you need
(optional) mode |`default`        | `default` for regular chiquito fistrum style and `latin` for the latin version
